# Your Rhythm - Personal Rhythm Tracker

A Progressive Web App (PWA) for tracking energy, mood, cycle patterns, and productivity in tune with your natural rhythm.

## Features

- 📊 **Daily Tracking** - Log energy, mood, focus, and flow
- 🌙 **Cycle & Moon Phases** - Track your cycle alongside lunar phases
- 🧠 **Brain Dump** - Free-form thought capture
- ⏱️ **Task Timer** - Time-box work with focus sessions
- 📅 **Calendar Integration** - Connect Notion, Google Calendar, Outlook
- 📈 **Insights** - Visualize patterns over time
- 📱 **Installable** - Add to home screen on any device
- 🔒 **Private** - All data stays local on your device

## Installation for Users

### Mobile (iPhone/Android):
1. Open the app URL in Safari (iOS) or Chrome (Android)
2. Tap "Add to Home Screen" or install prompt
3. App appears on home screen like a native app

### Desktop:
1. Visit the URL in Chrome or Edge
2. Click install icon in address bar
3. App opens in its own window

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deploy to Vercel

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel
```

Or push to GitHub and import to Vercel dashboard.

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS v4
- Vite
- Recharts (data visualization)
- Motion (animations)
- Lucide Icons
